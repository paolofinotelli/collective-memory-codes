# collective-memory-codes
This repository contains code for modeling and simulating **collective memory formation** and **distortion** in networked populations. The primary focus is on using mathematical models, such as **DeGroot consensus dynamics**, to explore how individual memory states evolve through social interactions within groups. The project provides Matlab scripts for simulating and analyzing the emergence of collective memory.

## Features:
- Model individual and collective memory with **DeGroot consensus processes**
- Simulate memory dynamics using matrix-based update rules
- Analyze memory convergence, oscillations, and stability

## Dependencies:
- MATLAB
